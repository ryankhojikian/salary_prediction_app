


## Prerequisites

- Python 3.14 or higher
- [uv](https://github.com/astral-sh/uv) - Fast Python package installer
- [Ollama](https://ollama.ai/) - For running local AI models

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ryankhojikian/salary_prediction_app.git
   cd salary_prediction_app
   ```

2. **Install uv (if not already installed):**
   ```bash
   # On Windows
   winget install --id=astral-sh.uv 2> $null || (curl -LsSf https://astral.sh/uv/install.ps1 | powershell)
   ```

3. **Install Python dependencies:**
   ```bash
    uv sync
    uv add streamlit
    uv add supabase pandas python-dotenv joblib scikit-learn --no-build
   ```

4. **Install and setup Ollama:**
   - Download and install Ollama from [ollama.ai](https://ollama.ai/)

5. **Train the machine learning model:**
   ```bash
   uv run python train_model.py
   ```

6. **Setup environment variables:**
   Create a `.env` file in the project root with your Supabase credentials:
   ```
   SUPABASE_URL=your_supabase_url
   SUPABASE_KEY=your_supabase_anon_key
   ```

## Running the Application

The application requires three separate terminals to run all components:

### Terminal 1: Start Ollama with the Model
```bash
ollama run hf.co/unsloth/Llama-3.2-1B-Instruct-GGUF
```

### Terminal 2: Start FastAPI Backend
```bash
uv run uvicorn app:app --reload --host 0.0.0.0 --port 8001
```

### Terminal 3: Start Streamlit Frontend
```bash
uv run streamlit run streamlit_app.py
```

## Usage

1. Open your browser and navigate to the Streamlit app (usually http://localhost:8501)
2. Enter your job parameters (experience level, employment type, job title, company size)
3. Get your salary prediction along with a personalized career story generated by AI

## API Endpoints

- `GET /` - Health check
- `POST /predict` - Salary prediction with AI story generation

## Project Structure

- `app.py` - FastAPI backend application
- `streamlit_app.py` - Streamlit frontend interface
- `train_model.py` - Machine learning model training script
- `ds_salaries.csv` - Dataset for training
- `salary_model.pkl` - Trained machine learning model (generated by train_model.py)
- `encoders.pkl` - Label encoders for categorical features (generated by train_model.py)
- `.env` - Environment variables file (create with Supabase credentials)
- `pyproject.toml` - Project configuration and dependencies
